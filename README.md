# Brasswick
An A1111 like website for image generation website that uses ComfyUI as a backbone.

## Features
### Current
 - Generation of images
 - ComfyUI backend usage
 - Web page (I know, very unique)
 - Positive and negative prompts
 - Control over seed, seed control, CFG, image size, batch size, steps, sampler, scheduler and the model.
 - Default workflow (currently doesn't work with other ones)
 - Rad theme (togglable)
 - ETA of generation
 - Quick dimension selector
 - LoRA support (up to 3 currently)
 - Canceling generation
 - Mobile support (flawed at times)
 - Generation history since startup
 - Prompt tempaltes (save prompts and reuse them)
 - Keyboard shortcuts
 - ComfyUI erver control (Linux only) (Start/restart/stop)

### WIP
 - Multi user support (role based permissions and passwords)

### Possible
 - Auto complete
 - Custom workflows and UIs
 - Built in inpainting and outpainting
 - Built in Upscaler
 - Prompt queue
 - Docker image/other easier install methods
 - Optional per-user quotas and storage limits
 - API/CLI
 - Public / private galleries
 - Tagging, search & similarity search
 - Recommended settings per model
 - Plugins or something
 - Better logging
 - WD14 tagging
 - GGUF support
 

## Disclaimer
Do not expose this UI in public networks or port forward this, as for there may be security vulnarbilities and other people could generate content on your machine.
